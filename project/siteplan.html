Project Site Plan

Site Name

SereniTrack

This name blends the ideas of serenity and tracking, directly relating to the core of a Personal Wellness Tracker project

Purpose

For personal fitness tracking

This site empowers individuals to achieve holistic well-being by tracking wellness metrics, fostering positive habits, and offering personalized insights to support a balanced, healthy lifestyle.

Scenarios

For personal fitness tracking

  • How can I track my daily mood and activities to improve my mental well-being?
  • What are some simple and effective tips for maintaining a balanced lifestyle?

Color Schema

https://coolors.co/4a90e2-a3d9a5-d6c3fc-f8c3a3-f5f5f5

#1c5d99(Calm Blue): Primary color for branding elements (logo, header, footer, buttons)

#7ecf9b(Serene Green): Secondary color for content highlights and interactive elements

#d6c3fc(Soft Lavender): Accents to provide depth and variety in UI components (e.g., hover effects)

#f8c3a3(Warm Peach): Attention-grabbing elements like alerts or motivational tips

#ffffff(Neutral color): Clean and unobtrusive background for most content sections

#333333(Dark color): Grey color for texts sections

Typography

Primary Font: Open Sans

  • Body text for all pages, ensuring high readability for longer paragraphs.
  • Form labels, input fields, and small informational text.

Secondary Font: Lora

  • Headings to create an elegant and modern look.
  • Motivational quotes and daily tips on the Home Page for emphasis.

Accent Font: Roboto Mono

  • Highlighted elements like data logs, statistics, or charts on the Tracker Page.
  • Code-like elements or small annotations in resources for a tech-savvy feel.
